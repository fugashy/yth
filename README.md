# blindpy

Process video file with YOLO.

Keep blind faces, car numbers, etc...

# environment

macOS (apple silicon)

python 3.10 or later

# how to use

```bash
pip install .

# 2 steps to create
# detect objects and output them as csv
blindpy yolo VIDEO_FILE_PATH

# blind objects
blindpy seal VIDEO_FILE_PATH
```
